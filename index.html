Passenger Counter

This handy counter is for people who work in stations and have to count passengers boarding vehicles.

I added an undo button to kick off any unruly groups that enter the carriage. It also resets the people entering if you entered too many and want to restart that count.

JavaScript handles all the calculations using functions when the buttons are clicked.

An array is used to keep a log of all the groups of passengers.

The HTML is updated using the DOM (Document Object Model).
